Plugins als Buttons

Um Plugins schnell zu erreichen, können Sie Ihre wichtigsten Plugins zur Buttonleiste hinzufügen.

Gehen Sie dabei wie folgt vor:

Öffnen Sie mit der Tastenkombination Strg-S bzw. über das Menü TV-Browser --> Einstellungen den Einstellungen-Dialog.

Klicken Sie auf den Karteireiter Plugins und wählen anschließend ein Plugin aus.

Markieren Sie nun das Kontrollkästchen Plugin zu Buttonleiste hinzufügen.

Das Plugin ist nun auch über die Buttonleiste verfügbar.

Hinweis: Wenn sich das Kontrollkästchen nicht markieren läßt, so steht diese Funktion für dieses Plugin nicht zur Verfügung.
