Adding channels

To add new channels to the program table, do the following:

Open the settings dialog by pressing Ctrl-S or by choosing Settings from the TV-Browser menu.

Activate the Channels tab and highlight the desired channels in the left area (use the Ctrl key to select multiple channels). Finally press the button showing an arrow pointing right to add the channels.

Hint: After having added new channels you should update the TV data to retrieve the program data for the new channels. Press F5 to do so.
