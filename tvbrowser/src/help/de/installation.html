Installation

Auf Windows-Systemen wird TV-Browser mit dem Installer-Programm installiert. Auf den anderen Systemen muß das .zip-File entpackt werden.

TV-Browser wird (auf Windows-Systemen) gestartet, in dem Sie die Datei "tvbrowser.exe" doppelklicken.
An der Kommandozeile startet man TV-Browser mit java -jar tvbrowser.jar

Bei Problemen mit der Installation lesen Sie bitte die FAQ (häufig gestellte Fragen) oder stellen Sie Ihre Frage im TV-Browser-Forum.

Eine TV-Browser-Installation besteht aus den folgenden Dateien und Verzeichnissen:

Installationverzeichnis

help/ Die Hilfe-Dateien
help/default/ Englischsprachige Version
help/de/ deutschsprachige Version
imgs/ die Bilder
log/ Die Protokoll-Dateien. Dieses Verzeichnis kann jederzeit gelöscht werden
plugins/ In diesem Verzeichnis befinden sie die Plugins. Wenn Sie Plugins aus dem Internet herunterladen, verschieben Sie sie in dieses Verzeichnis. TV-Browser wird sie automatisch erkennen.
themepacks/ In diesem Verzeichnis befinden sich die Skins für TV-Browser. Sie können die Skins aber auch in einem anderen Verzeichnis ablegen.
tvdata/ Hier legt TV-Browser die TV-Daten ab. In diesem Verzeichnis befindet sich für jedes installierte TV-DataService ein weiteres Unterverzeichnis, in dem TV-DataService-spezifische Dateien abgelegt werden.
tvdataservice/ Hier werden die TV-DataServices abgelegt. Eine Standardinstallation besteht aus genau einem solchen TV-DataService, dem TvBrowserDataService, das die TV-Browser-eigenen TV-Daten lädt.

Homeverzeichnis

.tvbrowser/ Im Homeverzeichnis erzeugt TV-Browser das Verzeichnis .tvbrowser. Dort werden die Einstellungen gespeichert.
.tvbrowser/settings.prop Die Einstellungen von TV-Browser. Diese Datei kann mit einem Texteditor editiert werden.
.tvbrowser/xxx.xxx.dat Daten, die das Plugin xxx gespeichert hat (z.B. Erinnerungen, Lieblingssendungen usw.)
.tvbrowser/xxx.xxx.prop Einstellungen, die das Plugin gespeichert hat. Diese Datei kann mit einem Texteditor editiert werden.
.tvbrowser/xxx.xxx.service Einstellungen, die ein Tv-DataService gespeichert hat. Diese Datei kann mit einem Texteditor editiert werden.
