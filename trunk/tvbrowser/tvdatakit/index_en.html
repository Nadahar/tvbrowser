PROVIDING TV LISTINGS FOR TV-BROWSER

using the TV-Browser tools

Overview

Author: Martin Oberhauser
Version: 2.0 (April 2005)
Translated by: Lena Pammer

This document is a step-by-step tutorial for providing tv listings with our tools.
I've already written a pdf document that you can download from our web site. However, many people thought it was too complicated and used too many technical terms, so I attempted to write a longer version that is easier to understand - this document.

Important: Before you provide tv listings to the public you must get explicit permission to do so from the rightful owners of the listings!

The tutorial consists of several parts:

  • Part I presents the TV-Browser tools and their use. If you own webspace you may upload your tv listings using ftp. TV-Browser will be able to download them from the webspace.
  • Part II explains how you can use our tools to import and publish listings that come inthe Excel file format.
  • Part III shows how you can import listings from xml files and use them in TV-Browser. This part is aimed at those who know how to program, but do not know Java. In this case, the xml file serves as a go-between between the data collector you have developed and the TV-Browser tools.

The tutorial requires no programming skills whatsoever. All the tools contained in the downloadable version of the tutorial.

If you have questions or requests concerning the tutorials do not hesitate to post to our forum.
