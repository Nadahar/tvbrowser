Offline-Modus benutzen

TV-Browser wird standardmäßig im sogenannten Offline-Modus betrieben.
Das bedeutet, es wird nur dann eine Verbindung mit dem Internet aufgebaut, wenn TV-Browser über die Aktualisierungsfunktion neue TV-Daten laden soll. Wenn Sie überkeine permanente Internetverbindung verfügen, sollten Sie TV-Browser im Offline-Modus betreiben.

Datumsangaben, die grau (statt schwarz) dargestellt werden, lassen sich im Offline-Modus nicht auswählen.

Hinweis:

Selbstverständlich können Sie auch im Offline-Modus über die Aktualisierungsfunktion TV-Daten herunterladen.
