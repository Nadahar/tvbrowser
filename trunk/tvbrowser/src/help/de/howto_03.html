TV-Daten auf einen anderen Rechner übertragen

Um TV-Daten auf einen anderen Rechner zu übertragen gehen Sie wie folgt vor:

Klicken Sie im Menü TV-Daten --> Exportieren und wählen Sie Name und Ort der Datei, in die die TV-Daten gespeichert werden sollen.

Kopieren Sie diese Datei dann auf den anderen Rechner. Starten Sie auf diesem Rechner ebenfalls TV-Browser und wählen Sie im Menü TV-Daten --> Importieren. Geben Sie die exportierte Datei an.

Hinweis: Es können nur jene Daten exportiert werden, die sich auf dem Rechner befinden.
Möglicherweise fehlen dem importerenden Rechner Senderdaten, über die der exportierende nicht verfügt.
