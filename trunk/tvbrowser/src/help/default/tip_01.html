Plug-ins as buttons

To quickly activate important plug-ins you can add them to the tool bar.

Follow these instructions:

Open the settings dialog by pressing Ctrl-S or by choosing Settings from the TV-Browser menu. Activate the Plugins tab.

Click the tab Plugins and select the plug-in you wish to add.

Check the box Add plugin to toolbar

From now on the plug-in will be available in the tool bar.

Hint: If the check box is greyed out the plug-in doesn't support being added as a button.
