Copying the TV data to another computer

To copy the TV data to another computer follow these steps:

Choose Export from the TV data menu and enter the name and location of the file that will contain the TV data.

Copy this file to the target computer. Start TV-Browser there and activate Import from the TV data menu. Choose the previously exported file.

Hint: You can only export data residing on the source machine.
If the target computer has subscribed to more channels than the source computer data will be missing because the source computer never downloaded it.
