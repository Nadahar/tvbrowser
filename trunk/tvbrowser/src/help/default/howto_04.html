Plugins installieren

Plugins sind Dateien, mit der Endung .jar.

Wenn Sie ein Plugin aus dem Internet heruntergeladen haben, müssen Sie es in das Verzeichnis plugins Ihrer TV-Browser-Installation kopieren.

Öffnen Sie anschließ mit der Tastenkombination Strg-S bzw. über das Menü TV-Browser --> Einstellungen den Einstellungen-Dialog und klicken Sie auf Plugins.

Verschieben Sie nun das Plugin von der linken auf die rechte Seite.
Das Plugin ist nun installiert und steht zur Verfügung.

Hinweis: Informationen über neue Plugins finden Sie auf der Homepage von TV-Browser:

http://tvbrowser.sourceforge.net
