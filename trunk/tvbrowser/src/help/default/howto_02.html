Sender hinzufügen

Um weitere Sender zur Programm-Tabelle aufzunehmen gehen Sie wie folgt vor:

Öffnen Sie mit der Tastenkombination Strg-S bzw. über das Menü TV-Browser --> Einstellungen den Einstellungen-Dialog.

Klicken Sie dann auf Kanäle, markieren Sie die gewünschten Sender der linken Seite (bei gedrückter Strg-Taste) und drücken auf den Button mit dem Pfeil nach rechts, um die Sender aufzunehmen.

Hinweis: Anschließend sollten Sie eine TV-Daten-Aktualisierung durchführen, um die Daten der neuen Sender in der Programm-Tabelle sehen zu können.
Drücken Sie hierfür die Taste F5.
