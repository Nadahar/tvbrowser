
Diese Datei soll einen Überblick über die geplanten Änderungen in den nächsten Versionen liefern.
Um die Release-Zyklen klein zu halten, werden pro Version möglichst wenige "große" Änderungen durchgeführt. Ist die "große" Erweiterung implementiert, wird die (Beta-)Version freigegeben.



nächste geplante Release: 2.1 (Plugin-Schnittstelle)
=============================
- Verbesserungen der Plugin-Schnittstelle(n):
    * Die TV-Dataservice-Schnittstelle soll nun Sendergruppen kennen,
      damit der User die Sendergruppen-URL nicht mehr händisch eingeben muß
    * TV-Dataservices und Plugins sollten in *einem* Verzeichnis abgelegt werden können
    * Ein Plugin, das eine Exception wirft, soll gewaltsam beendet werden können und darf die
      Stabilität von TV-Browser keinesfalls beeinträchtigen
    Zugehörige Tracker-Items:
      1280109 TvDataService-Schnittstelle erweitern
      1277641 "Force Quit" für Plugins
      1032014 Plugin-System überarbeiten

- PrintPlugin:
    * Es sollen Filter gedruckt werden können
    * Markierte Sendungen sollen auch auf dem Papier markiert werden
    Zugehörige Tracker-Items:
      1265487 PrintPlugin: Drucken im Querformat nicht möglich
      1280122 PrintPlugin: Sendungen markieren
      1280121 PrintPlugin: Filter drucken


Release 2.2 (Grafische Verbesserungen)
===========
- Grafische Verbesserungen
    * Die Icons sollen austauschbar werden
    * Farben sollten veränderbar sein - wenn auch nur über ein Config-File
    * System mit nicht-Modalen Dialog bauen
    * Usability-Überlegungen
    Zugehörige Tracker-Items:
      1280129 Nicht-modale Dialoge verwenden
      1260108 Icon themes
      1280070 Farben ändern
